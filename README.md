# 🔷 HexOS — The OS for AI Agents

**Deploy intelligent agents that handle your business operations 24/7.**

HexOS is a multi-agent orchestration platform that lets you deploy AI agents for content, outreach, research, trading, development, and more — all from a single command line.

## Features

- 🤖 **Multi-Agent Orchestration** — Deploy specialized agents that coordinate automatically
- 🧠 **Persistent Memory** — Agents remember context across sessions (powered by Recall)
- 🛡️ **Safety Guardrails** — Built-in action validation with Vigil
- 📅 **Scheduling** — Heartbeats, cron jobs, and event-driven triggers
- 💬 **Multi-Channel** — Telegram, Discord, Slack, WhatsApp, Signal, and more
- 🔧 **Extensible** — Add custom skills, tools, and integrations
- 🏠 **Self-Hosted** — Your data stays on your server

## Quick Start

```bash
npm install -g @hexitlabs/hexos
hexos setup
hexos gateway start
```

## Health Check

```bash
hexos doctor
```
