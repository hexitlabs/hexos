# `hexos setup`

Initialize `~/.hexos/hexos.json` and the agent workspace.

Related:
- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
hexos setup
hexos setup --workspace ~/clawd
```

To run the wizard via setup:

```bash
hexos setup --wizard
```
